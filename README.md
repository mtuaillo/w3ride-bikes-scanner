# W3Ride sales scanner

Command tool to scan [W3 Ride](https://w3ride.io/) bikes currently on sale, and display best deal for each score.

![hello](./doc/output.png)

# Dependencies

- PHP 8.1
- SQLite extension
- Composer
- [Taskfile](https://taskfile.dev/)

# Installation

```shell
task install
```

# Usage

```shell
task list-sales
```
